# REAL-TIME-COLLABORATIVE-DOCUMENT-EDITOR

## PROJECT OVERVIEW

This project is a Real-Time Collaborative Document Editor developed using React.js, Node.js, Socket.IO, and MongoDB. Multiple users can edit the same document simultaneously and changes are synchronized in real time.

## FEATURES

- Real-Time Collaboration
- Live Document Editing
- Socket.IO Integration
- MongoDB Data Storage
- Responsive User Interface
- Full Stack Architecture

## TECHNOLOGIES USED

### Frontend
- React.js
- CSS3

### Backend
- Node.js
- Express.js
- Socket.IO

### Database
- MongoDB
- Mongoose
  
## HOW TO RUN

### Backend

cd server

npm install

npm start

### Frontend

cd client

npm install

npm start

## CONCLUSION

The application demonstrates real-time collaborative editing using Socket.IO and persistent storage using MongoDB.
